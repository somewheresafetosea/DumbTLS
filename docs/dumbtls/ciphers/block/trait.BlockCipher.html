Trait for a generic block cipher.

This trait is to be implemented by structs representing a block cipher algorithm: They must be able to encrypt a single block of data, and decrypt a single block of data. They must also be able to provide their block size.

It is structs that implement this trait that are operated on by the block cipher modes of operation in the block module.
